# Assignment 1 — Classical Reinforcement Learning

This assignment focuses on modeling and solving a predator-prey
gridworld environment using classical reinforcement learning methods.

## Parts

### Part 1 — Environment and Policy Modeling
Implementation of:
- simulator
- transition kernels
- reward functions
- policy evaluation

### Part 2 — Dynamic Programming
Implementation of:
- value iteration
- policy iteration
- induced policies

### Part 3 — Kernel Estimation
Implementation of:
- estimated transition kernels
- approximate value iteration
- comparison with exact kernels

## Environment
- Predator-prey gridworld
- Discount factor γ = 0.99
- Random prey movement

## Technologies Used
- Python
- NumPy
- Matplotlib
